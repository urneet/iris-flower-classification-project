# iris-flower-classification-project
Machine Learning project for Iris Flower Classification using Exploratory Data Analysis, Hypothesis Testing, Logistic Regression, KNN, Decision Tree models and Streamlit web application.
# Iris Flower Classification Project

## Project Overview
This project classifies Iris flowers into Setosa, Versicolor and Virginica species using Machine Learning algorithms based on sepal and petal measurements.

## Project Components
Part 1 – Research Report on Data Analytics, Data Science and AI.
Part 2 – Project Action Plan.
Part 3 – Exploratory Data Analysis and Hypothesis Testing.
Part 4 – Model Building and Evaluation.
Part 5 – Streamlit Web Application.
Part 6 – GitHub Repository.
